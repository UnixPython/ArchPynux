# ArchPyNux

🚧 **Project in development** 🚧

## 📢 We are looking for contributors

We are building a team to create **ArchPyNux**, an **Arch Linux-based distribution** focused on:

* 🔐 Security and hardening
* 🐍 Python development
* 🖥️ Fully TTY-based environment (no graphical interface)

Our goal is to build a lightweight, secure system optimized for developers and security enthusiasts who prefer working directly from the terminal.

## 🚀 Project Status

The project is currently in an early stage.
Very soon we will be publishing:

* 📦 Base system files
* 🧩 Related projects
* 💻 Source code

This will allow the community to start contributing actively to the development of ArchPyNux.

## 🤝 Want to contribute?

We would love to have you on board!
We are looking for people interested in:

* Cybersecurity
* Python development
* Linux / Arch Linux systems
* Terminal optimization and automation

It doesn’t matter if you are a beginner or an expert — every contribution counts 🙌
